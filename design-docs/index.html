JavaScript (Alpine.js) 
window.newsCarousel = () => ({
    currentIndex: 0,
    progress: 0,
    isPlaying: true,
    timer: null,
    itemDuration: 5000,
    
    newsItems: [
        {
            id: 1,
            title: 'Your news title here',
            excerpt: 'Brief description of the news item',
            image: 'path/to/your/image.jpg',
            timeAgo: '2 hours ago'
        }
        // Add more news items...
    ],
    
    init() {
        this.start();
    },
    
    start() {
        if (this.timer) clearInterval(this.timer);
        this.isPlaying = true;
        this.progress = 0;
        
        this.timer = setInterval(() => {
            if (this.isPlaying) {
                this.progress += 2;
                if (this.progress >= 100) {
                    this.next();
                }
            }
        }, 100);
    },
    
    next() {
        this.currentIndex = (this.currentIndex + 1) % this.newsItems.length;
        this.progress = 0;
    },
    
    select(index) {
        this.currentIndex = index;
        this.progress = 0;
    },
    
         toggle() {
         this.isPlaying = !this.isPlaying;
         if (!this.isPlaying && this.timer) {
             clearInterval(this.timer);
         } else if (this.isPlaying) {
             this.start();
         }
     },
     
     reset() {
         this.currentIndex = 0;
         this.progress = 0;
         this.start();
     }
});

Configuration Options

itemDuration

Duration in milliseconds for each carousel item auto-advance

Type: Number | Default: 5000

newsItems

Array of news objects containing carousel content

Type: Array<NewsItem> | Required: true
NewsItem properties: id, title, excerpt, image, timeAgo

Available Methods

toggle()

Play/pause auto-rotation

select(index)

Jump to specific item

next()

Advance to next item

start()

Start/restart auto-rotation

reset()

Reset to first item and restart

Implementation Notes

Requires Alpine.js and Lucide Icons to be loaded
Auto-rotation can be controlled via user interaction
Responsive design works on mobile and desktop
Progress bar indicates current item timing
